# Wracket

My [Pocket](https://getpocket.com) client on Web.

- Pocket API: https://getpocket.com/developer/docs/overview

## Development

- Backend: Ruby on Rails (for now)
- Database: PostgreSQL
- Frontend: React and Redux in TypeScript
- CI/CD: CircleCI
- Server provisioning: Ansible
- Server: GCE (for now)
- TLS: Let's Encrypt
